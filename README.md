##How to use

1. Move the files to your home directory
2. Rename the files to be hidden:
`mv bash_profile .bash_profile`
`mv git-completion.sh .git-completion.sh`
`mv git-parse-branch.sh .git-parse-branch`
3. Reload .bash_profile
`source ~/.bash_profile`

Enjoy!
